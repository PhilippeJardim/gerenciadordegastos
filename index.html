Integração com API Pública

A aplicação passa a consumir uma API externa para exibir a cotação atual do dólar em real, agregando valor financeiro ao controle de gastos.

Cotação do dólar

Esta área recebe os dados enviados pela view Django através da variável cotacao_dolar. Se a API estiver indisponível, a página mostra uma mensagem amigável sem quebrar o sistema.

{% if cotacao_dolar %} API carregada com sucesso

{{ cotacao_dolar.nome }}

R$ {{ cotacao_dolar.valor }}

Atualizado em: {{ cotacao_dolar.data }}

{% else %} Cotação indisponível

Não foi possível carregar a cotação do dólar no momento. Tente novamente mais tarde.

{% endif %}

API escolhida

AwesomeAPI Economia
Endpoint utilizado: USD-BRL
Método: HTTP GET
Finalidade: consultar a cotação do dólar em real.

Funcionalidades da aplicação

O Gerenciador de Gastos permite controlar despesas, visualizar totais, filtrar categorias e agora acompanhar dados financeiros externos.

Adicionar despesas

Registro de novos gastos com data, valor, descrição e categoria.

Listar despesas

Visualização organizada das despesas cadastradas na aplicação.

Filtrar por categoria

Consulta rápida de despesas por alimentação, transporte, lazer e outras categorias.

Editar despesas

Atualização dos dados de uma despesa já cadastrada.

Excluir despesas

Remoção de registros com confirmação de segurança.

Cotação externa

Consulta de dados reais da internet usando API pública de economia.

Roadmap da entrega

Sequência recomendada para concluir e apresentar a etapa intermediária.

Passo 1

Abrir Issue

Registrar no GitHub a demanda de integração da API de cotação do dólar.

Passo 2

Criar branch

Criar e trabalhar na branch entrega-intermediaria, sem alterar diretamente a main.

Passo 3

Programar API e teste

Criar o serviço em Python, mostrar os dados no template e validar com pytest.

Passo 4

Deploy e PR

Publicar a aplicação, atualizar o README com o link e abrir Pull Request fechando a Issue.

Comandos principais

Use estes comandos no terminal do VS Code para registrar e enviar a entrega. 

git checkout main
git pull origin main
git checkout -b entrega-intermediaria

pip install -r requirements.txt
pytest
ruff check .
python manage.py runserver

git add .
git commit -m "Integra cotacao do dolar com API publica"
git push -u origin entrega-intermediaria

Entrega pronta para validação

Depois do push, abra o Pull Request no GitHub usando a descrição com Closes #número-da-issue.
